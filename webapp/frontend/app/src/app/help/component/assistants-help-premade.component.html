Code Review Bot

A senior software engineer on a mission to make your code cleaner, safer, and more efficient. The bot reads the code you paste, then produces a structured review that covers everything from style and readability to performance and security.

  • Highlights strengths and best practices.
  • Pinpoints bugs, logical errors, and potential edge cases.
  • Assesses readability, maintainability, and performance.
  • Flags security concerns (e.g., injection risks, improper error handling).
  • Provides actionable suggestions with line-number references.

How to use: Paste the code into the prompt, then let the bot return a concise review.

Mermaid Diagram Bot

An expert in crafting and refining Mermaid.js diagrams. The bot ensures your diagrams are accurate, visually balanced, and syntactically correct.

  • Verifies clarity of the request before generating a diagram.
  • Asks targeted clarifying questions when the prompt is ambiguous.
  • Chooses the most suitable Mermaid diagram type (flowchart, sequence, class, state, ER, etc.).
  • Produces clean, self-contained Mermaid code blocks.
  • Provides a plain-text copy of the code for easy copy-paste or version control.

How to use: Submit a description of the diagram you need; if more detail is required, the bot will ask follow-up questions before delivering the final Mermaid code block.
