Documents Management

The Documents page lets you upload, view, filter, and delete documents that can be used by your AI assistants. Each document can be attached to one or more assistants, so the page also shows which assistants are currently using it.

Adding a New Document

Click Add Document to open the editor:

  • Title - the name shown in the list.
  • File - the actual file to upload (any format the backend accepts).

After filling out the fields, click Submit to submit. The document will be processed asynchronously - you'll see a Processingâ€¦ indicator until the backend finishes indexing it.

If you change your mind, click Cancel to cancel.

Editing a Document

You can't edit a document. You'll have to delete it and re-upload it if the contents have changed and you want that reflected in your Assistants.

Filtering

Use the Filter input to search by title. The filter is case-insensitive and updates the list in real time as you type.

Document Status

A red Processing... or Document Failed to Process! message appears if the document is still being processed, or if something went wrong. Don't use the document for RAG until it's finished processing or you might get strange results.

Ownership & Deletion

Only the owner of a document can delete it. The delete button (trash icon) appears in the top-right corner of each owned document.

Using Documents in Assistants

When a document is attached to an assistant, the assistant can retrieve the content during a conversation. The list shows a badge for each assistant that uses the document. This helps you keep track of dependencies and avoid orphaned files.

Quick Start

  1. Click Add Document.
  2. Enter a title and choose a file.
  3. Click Submit.
  4. Wait for the document to appear in the list as READY.
  5. Attach the document to an assistant via the assistant editor.
