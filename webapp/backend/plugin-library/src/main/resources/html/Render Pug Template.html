Render Pug Template Output Task

Turn the raw results of your workflow into a polished HTML page using a Pug template. The rendered page is returned as the final output of the workflow.

What It Does

  • Accepts the ExecutionResult from your workflow.
  • Feeds that data into a Pug template you supply.
  • Produces a single text/html file containing the fully rendered page.

Configuration

The task requires a single configuration setting:

Parameter Type Description
template String The Pug template to render.
See the Pug Syntax Help page for more help on Pug. Consider letting Minty write your Pug template!

Accessing Data in Your Pug Template

The template can reference these objects:

  • startTime – Time the workflow started.
  • endTime – Time the workflow completed.
  • results – map of step names to lists of result objects.
  • errors – map of step names to lists of error messages.

Output Format

The task always produces output in text/html.

Summary

Use the Render Pug Template output task whenever you need to convert workflow results into a clean HTML report or web page. Give it a Pug template and let the task handle the rendering.
